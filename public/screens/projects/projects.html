Project 1

ERP Application:

In this project, I developed an ERP (Enterprise Resource Planning) application that covers all processes from the material purchasing stage of a product to the final sale. I implemented this application using Flutter and Firebase. The application includes many important features such as tracking shipments, placing orders, recording sales, and graphs that calculate monthly net income. There are also special pages that can keep track of employees. When logged in as admin, it is possible to perform all operations. Separate user login information and passwords are defined for shippers, product preparers and other users. Users access certain pages based on their authority, thus providing a customized experience for each role.

Project 2

Custom Mobile Application for Factory: In this project, I developed a mobile application specific to a factory using Flutter. The application includes features such as shopping, adding products to the cart, adding to favorites, management via the admin panel, and profile updating. Users can view and purchase products by logging in with their email and password. Also, I integrated this mobile app with the desktop app. For this integration, I created APIs using MySQL and ensured data synchronization between the two platforms. In this way, users can enjoy the same experience on both mobile and desktop.

Project 3

Advanced Shopping Mobile Application: This shopping mobile application, which I developed using Flutter and Firebase, includes all the features found in a classic shopping application. In this project, I focused specifically on code organization and user experience. With the experience I gained for factory performance. Data such as product sales analyses, inventory management and financial reports can be viewed and analyzed instantly by users. Mobile and Web Compatible: It offers seamless operation on both mobile application and web browser. In this way, managers and employees can access anytime and anywhere. This ERP solution is designed to increase the efficiency of businesses and improve decision-making processes, with a focus on optimizing factory management processes. With its user-friendly interface and reliable structure, it helps factories manage their daily operations effectively.
